2026.01.01. 기준
📖 사용방법
공사 원가계산 간접비 요율 검증기
지방자치단체 입찰·계약집행기준에 따라 공사 견적서의 간접비 항목이 요율 최고가 이내인지 자동으로 검증합니다.
📋 탭 1 — 견적서 검증
① 견적서 파일 업로드 (선택)
  • .xlsx / .xls: 엑셀 견적서를 드래그하거나 클릭하여 업로드하면 금액이 자동으로 입력됩니다.
  • .pdf: 텍스트 레이어가 있는 PDF는 자동 추출됩니다. 스캔 이미지 PDF는 수동 입력이 필요합니다.
  • 파일 없이 아래 입력란에 직접 숫자를 입력해도 됩니다.
② 공사 조건 설정
  • 직접공사비 규모: 해당 공사의 직접공사비 금액 범위 선택
  • 공사기간: 계약상 공사기간 선택
  • 공사구분: 건축 또는 산업설비(건축) 선택
  • 추정가격: 일반관리비·이윤 요율 적용 기준 금액 범위 선택
  • 사회보험료: 고용보험료 등급 및 산재보험료율 확인 후 입력
  • 관급금액: 도급자설치 관급 금액이 있는 경우 입력, 없으면 0
③ 기초 원가 자료 입력
  • 직접노무비, 재료비, 경비를 입력합니다. (파일 업로드 시 자동 입력)
④ 견적서 계상 금액 입력
  • 견적서에 기재된 각 간접비 항목 금액을 입력합니다. (파일 업로드 시 자동 입력)
  • 해당 없는 항목은 0으로 두면 됩니다.
⑤ 요율 검증 실행
  • ✓ 적정: 견적 계상액이 요율 최고가 이내
  • ▲ 초과: 요율 최고가를 초과 → 감액 필요
  • 미계상: 해당 항목이 견적서에 없음
  • 결과 화면에서 🖨️ 인쇄 / PDF 저장 버튼으로 출력 가능
🔍 탭 2 — 요율 조회
  • 조건별 즉시계산: 금액 입력 시 각 항목 계산액을 즉시 산출
  • 나머지 탭: 간접노무비·기타경비, 일반관리비·이윤, 건설기계대여대금, 산업안전보건관리비, 사회보험료, 기타 요율표를 전체 조회

간접노무비 · 기타경비 요율표

건축=산업설비(건축) 동일 / 건축제비율(26.1.1.) 시트
• 간접노무비: (직노) × 율  •  기타경비: (재+직노+간노) × 율
• 기타경비 항목: 수도광열비, 복리후생비, 소모품비 및 사무용품비, 여비·교통·통신비, 세금과공과, 도서인쇄비
• 산업설비(건축) 해당공종: 지열공사, 소각시설 등

일반관리비 · 이윤 요율표

건축제비율(26.1.1.) 시트
• 일반관리비: (재+노+경) × 율  •  이윤: (노+경+일) × 율

건설기계대여대금 지급보증액 발급금액

건축제비율(26.1.1.) 시트
• 산출식: (직접공사비) × 율
• 「지방계약법 시행령」 제51조 제1항 적용 의무 대상: 추정가격 300억↑ 공사, 대형공사계약, 특정공사, 기술제안 입찰

산업안전보건관리비 요율표

건축제비율(26.1.1.) 시트
• 도급자관급 미포함: (재+직노) × 율 + 기초액
• 도급자관급 포함: (재+직노+관급) × 요율② + 기초액 과 [(재+직노) × 요율③ + 기초액] × 1.2 중 작은 금액
• 고용노동부 고시 제2024-53호 (2024.9.19.) / 총 공사금액 2천만원 이상 건설공사 적용

사회보험료 요율표

건축제비율(26.1.1.) 시트
• 건강·노인장기요양·연금보험료: 공사기간 1개월 이상 건설공사
• 산재보험료: 노동부 고시 제2024-77호(2024.12.30.)
• 사회보험료 적용기준: 국토부 고시 제2021-905호 (2021.7.1.)

환경보전비

(직접공사비) × 율
• 환경관리비의 산출기준 및 관리에 관한 지침 (국토부 고시 제2018-528호, 2018.8.30.)
• 적용제외: 전기·정보통신·소방시설공사, 국가유산 수리공사

건설하도급대금지급보증서발급수수료

(직접공사비) × 율
• 국토부 고시 제2016-921호 (2016.12.19.) / 적용제외: 전문공사, 국가유산수리공사

공사이행보증수수료

공사규모(직접공사비) 기준
• 「지방계약법 시행령」 제51조 제1항 적용 의무 대상
• 퇴직공제부금비: (직노) × 2.3% — 추정금액 1억 이상 건설공사 (국토부 고시 제2015-610호, 2015.8.20.)
