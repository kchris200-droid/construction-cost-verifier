# 공사 원가계산 간접비 요율 검증기

지방자치단체 입찰·계약집행기준에 따라 공사 견적서의 간접비 항목이 요율 최고가 이내인지 자동으로 검증하는 도구입니다.

**적용 기준**: 2026년 1월 1일 기준 건축·산업환경설비공사

---

## 주요 기능

- 엑셀(.xlsx/.xls) 또는 PDF 견적서 업로드 시 금액 자동 추출
- 간접노무비, 기타경비, 일반관리비, 이윤, 건설기계대여대금, 산업안전보건관리비, 사회보험료(건강·연금·고용·산재) 항목별 요율 검증
- 요율 초과 항목 즉시 확인 및 차액 산출
- 결과 인쇄 / PDF 저장 지원
- 요율 조회 탭에서 항목별 요율표 전체 열람 및 조건별 즉시 계산

---

## 사용 방법

### 탭 1 — 견적서 검증

1. **파일 업로드** (선택): 엑셀 또는 PDF 견적서를 드래그하거나 클릭하여 업로드
   - 파일 없이 아래 입력란에 직접 숫자를 입력해도 됩니다
   - PDF는 텍스트 레이어가 있는 경우에만 자동 추출됩니다 (스캔본은 수동 입력)
2. **금액 확인**: 자동 불러온 금액이 견적서 원본과 일치하는지 반드시 확인 후 수정
   - 불러오지 못한 항목은 직접 입력
3. **공사 조건 설정**: 직접공사비 규모, 공사기간, 공사구분, 추정가격, 사회보험료 등급 등 선택
4. **기초 원가 자료 입력**: 직접노무비, 재료비, 경비 입력 (파일 업로드 시 자동 입력)
5. **견적서 계상 금액 입력**: 견적서에 기재된 각 간접비 항목 금액 입력 (파일 업로드 시 자동 입력)
6. **요율 검증 실행** 클릭

검증 결과:
- **적정**: 견적 계상액이 요율 최고가 이내
- **초과**: 요율 최고가 초과 → 감액 필요
- **미계상**: 해당 항목이 견적서에 없음

### 탭 2 — 요율 조회

- **조건별 즉시계산**: 금액 입력 시 각 항목 계산액 즉시 산출
- 항목별 요율표(간접노무비·기타경비, 일반관리비·이윤, 건설기계대여대금, 산업안전보건관리비, 사회보험료, 기타) 전체 열람

- 지방계약법 제11조, 시행령 제8조~10조, 시행규칙 제6조

## 기술 스택

순수 HTML/CSS/JavaScript 단일 파일로 구성되어 있으며, 별도 서버나 설치 없이 브라우저에서 바로 실행됩니다.

- [PDF.js](https://mozilla.github.io/pdf.js/) v3.11.174 — PDF 텍스트 추출
- [SheetJS](https://sheetjs.com/) v0.18.5 — 엑셀 파싱

업로드한 견적서 파일은 사용자 브라우저 내에서만 처리되며 외부 서버로 전송되지 않습니다.
